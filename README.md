# Hive-Connect
## https://aaubrreyyy.github.io/Hive-Connect/
Prpject Description:Hive Connect is the repository name for The Hive's CINS 5318 - Software Engineering group project. Hive Connect is a volunteer website in which The Hive aims to connect Prairie View A&amp;M University students with local organizations seeking volunteers! 
Installation Instructions:
  Prerequisites: 
  - A web browser (Chrome, Edge, Firefox,etc
  - GitHub account
  - Can use GIT CMD to serve as a clone to the repository
  - Code editor(VS, Notebook++, etc)
Installaion
-Go to repository on GitHub
-Select the green "code" button
-Select download Zip
-Unzip the folder on your computer
-Open folder and double click "index" to open in your browser"
Options: Run website locally in Browser or Server(VS Code, etc)
